Cheat sheet VIM

Déplacement Commande

Déplacement dans VIM :

  • k
    Haut.
  • l
    Droite.
  • h
    Gauche.
  • j
    Bas.
  • G
    Se déplacer en bas du fichier.
  • gg
    Se déplacer en bas du fichier.
  • (chiffre)+G
    Se déplacer sur une ligne précise.
  • ctrl-f
    Descendre d'un écran.
  • ctrl-b
    Remonter d'un ecran.
  • b
    Aller au début du mot précédent.
  • w
    Aller au début du mot suivant.
  • e
    Aller a la fin du mot suivant.

Commande de VIM :

  • :wq
    Enregistrer le fichier et quitter.
  • :q!
    Quitter le fichier sans enregistrer.
  • x
    Supprimer le caracrtère sous le curseur.
  • i
    Insérer du texte sous le curseur.
  • a
    Insérer du texte aprés le curseur.
  • dw
    Supprime le mot ( Curseur placer sur la 1er lettre du mot).
  • d$
    Supprimer jusqu'au bout de la ligne ( a partir du curseur).
  • dd
    Supprimer toute la ligne.
  • u
    Annuler les dernières commandes.
  • p
    Place aprés le curseur ce qu'on vient de supprimer.
  • r(+1 lettre)
    Remplace la lettre sous le curseur par le caractère choisit.
  • cs
    Supprime le mot pour le réecrire.
  • c$
    Supprime le reste de la ligne pour la réecrire.
  • CTRL-G
    Afficher la position dans le fichier et son etat actuel.
  • /
    Fonction recherche.
  • %
    Placer sur des "(,[,{" cette commande sert a trouver la fermeture de la parenthése sur laquel le curseur et placer.
  • ?
    Recherche dans le sens oposer de la commande "/".
  • :!
    Permet d'effectuer n'importe quel commande externe ( comme sur le terminal) exemple = :!ls.
  • :w
    Permet d'enregistrer un fichier.
  • v
    Selectionner du texte.
  • :r(nomdefichier)
    Permet de coller le contenue d'un fichier sur le fichier actuel.
  • o
    Passer en mode insertion sur la ligne dessous le curseur.
  • R
    Remplacer plusieurs caractères.
  • y et p
    En mode visuel (v) utiliser "y" pour copier les élement selectionner et "p" pour les coller.
  • :help
    Aide de VIM.
  • :e ~/.vimrc
    Acceder au fichier config de VIM.
